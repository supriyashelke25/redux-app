# 📝 Todo App (Redux)

A simple and responsive Todo Application built using **React** and **Redux** for state management.  
This application helps users manage daily tasks efficiently with a clean and user-friendly interface.

🔗 **Live Demo:** https://myredux-app.netlify.app/

---

## 🚀 Features

- ➕ Add new tasks  
- ❌ Delete tasks  
- ✅ Mark tasks as completed  
- 🔄 Centralized state management using Redux  
- 📱 Responsive design  
- ⚡ Fast and lightweight application  

---

## 🛠️ Tech Stack

- ⚛️ React JS  
- 🗂️ Redux (Redux Toolkit)  
- 🎨 CSS / Bootstrap  
- 🌐 Netlify (Deployment)
